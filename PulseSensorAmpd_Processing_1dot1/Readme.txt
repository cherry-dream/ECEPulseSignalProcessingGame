Game Name: Heartbeat Running

How to Run:
Open PulseSensorAmpd_Processing_1dot1.pde in processing,with your controller all set up.
Compile and run to enjoy.

How to play: 
1.Put your finger upon the pluse sensor. 
2.Your character will start running automatically once the sensor begins picking up your heartbeat.
3.The faster your heartbeat, the easier it is to out run the boulder
4.But if you outrun the boulder for too long, it will get faster and faster
